# 酒店SAAS平台需求清单

> **生成时间**: 2025-11-27
> **用途**: 功能报价清单（可直接复制到Excel）
> **说明**: 仅包含已实现的功能，按菜单结构组织

---

## 平台后台

| 一级菜单 | 二级菜单 | 三级功能（用户故事） |
|---------|---------|-----------------|
| 酒店入驻 | 加盟申请 | 支持查看和筛选酒店加盟申请，可跟进处理并导出申请记录 |
| 酒店入驻 | 合作酒店 | 支持查看合作酒店列表，可编辑酒店信息、设置上下线状态、配置平台抽佣比例 |
| 酒店入驻 | 协议模板 | 支持创建和编辑合作协议模板，可进行版本控制 |
| 酒店入驻 | 签约记录 | 支持查看酒店签约记录，可筛选和导出 |
| 酒店管理 | 房态总览 | 支持查看7天房态日历，可通过颜色标识区分房间状态（可用/已订/已住/脏房/维修），显示可用房数统计 |
| 酒店管理 | 预订管理 | 支持查看预订列表并按状态筛选，可新增前台预订，查看预订详情 |
| 酒店管理 | 入住管理 | 支持查看待入住和在住客人，可办理入住和退房，支持续住和押金管理 |
| 酒店管理 | 房价管理 | 支持查看房价日历并内联编辑价格，可设置周末节假日差价，支持批量设置价格 |
| 酒店管理 | 客房管理 | 支持查看房间列表并按房型楼层筛选，可配置房间设施，管理房间状态（可用/维修/停售） |
| 订单管理 | 订单列表 | 支持查看订单列表并按状态筛选，可查看订单详情并导出 |
| 订单管理 | 退款管理 | 支持审核退款申请，可同意或拒绝，查询退款记录 |
| 争议处理 | 退款申请 | 支持查看和筛选退款申请，可审核处理，支持仲裁流程 |
| 积分管理 | 积分规则配置 | 支持配置注册奖励积分、邀请奖励积分、积分兑换汇率、最大抵扣比例、积分过期规则 |
| 积分管理 | 积分调整 | 支持按用户ID搜索用户，可手动发放或回收积分，查看用户积分流水 |
| 积分管理 | 积分增值服务 | 支持配置环保奖励服务项和积分换购服务项，可新增编辑删除，设置启用禁用状态 |
| 会员管理 | 会员等级设置 | 支持配置VIP0-9各等级的升级所需间夜数、保级所需间夜数、会员有效期 |
| 会员管理 | 会员邀请记录 | 支持查看会员邀请记录并筛选，可按用户/商户邀请分类查看，支持三级菜单展示 |
| 用户管理 | 用户列表 | 支持查看用户列表并筛选，可查看用户详情 |
| 系统管理 | 账号列表 | 支持创建和编辑平台管理员账号，可配置权限 |
| 系统管理 | 用户列表 | 支持查看和管理用户信息 |
| 系统管理 | 协议配置 | 支持编辑用户协议和隐私政策，可进行版本控制 |
| 系统管理 | 标签配置 | 支持创建和编辑系统标签，可查看标签使用统计 |

## 酒店后台（商户端）

| 一级菜单 | 二级菜单 | 三级功能（用户故事） |
|---------|---------|-----------------|
| 订单管理 | 订单列表 | 支持筛选和搜索订单，可查看订单详情、取消订单、导出订单数据 |
| 订单管理 | 订单日历 | 支持月视图查看订单分布，可查看入住率，按条件筛选并导出 |
| 订单管理 | 客诉退款 | 支持查看退款申请，可审核并同意或拒绝 |
| 订单管理 | 用户评价 | 支持查看用户评价并筛选，可回复评价并导出 |
| 收益管理 | 房价日历 | 支持查看房价日历，可单日设置价格或批量设置 |
| 收益管理 | 库存日历 | 支持查看库存日历，可单日设置库存或批量设置 |
| 门店配置 | 基本信息 | 支持查看门店身份信息（锁定），可编辑联系方式、上传Logo、设置推荐标签、编辑门店介绍、上传列表封面、视频素材、最新情报图片 |
| 门店配置 | 门店设施 | 支持选择亮点标签（10个选项），可勾选交通服务、清洁服务、安全安保、运动设施、康体设施、无障碍设施 |
| 门店配置 | 门店图片 | 支持上传小程序分享图并编辑文案，可上传主图（最多5张）并排序删除 |
| 门店配置 | 周边信息 | 支持按类别（景点/交通/餐饮/购物/医疗）分组管理周边位置，可新增编辑，支持拖动排序 |
| 门店配置 | 酒店政策 | 支持配置预订时间、取消规则、入住年龄限制、儿童政策、宠物政策、押金政策、支付方式 |
| 门店配置 | 早餐政策 | 支持设置是否提供早餐，可配置早餐类型、菜系、供应时间、加餐价格、儿童收费规则列表 |
| 门店配置 | 加床政策 | 支持按院落分组展示房型，可配置加床和婴儿床政策（床型/数量/价格） |
| 门店配置 | 非房产品 | 支持添加和编辑非房产品，可设置价格和库存 |
| 房型管理 | 房型列表 | 支持查看房型列表，可创建、编辑、删除房型 |
| 房型管理 | 房型图片 | 支持上传房型图片，可排序并设置主图 |
| 房型管理 | 房间管理 | 支持添加和编辑房间，可管理房间状态 |
| 老客服务 | 代客下单 | 支持填写订单信息，可限制购买人、生成专属二维码、设置特惠价格 |
| 老客服务 | 邀请会员 | 支持点击生成邀请二维码，可赠送VIP1-3体验会员（期限7天） |
| 积分服务 | 服务配置 | 支持查看平台定义的积分服务项（环保奖励/增值服务），可启用或禁用本店支持的服务 |
| VIP折扣 | 折扣配置 | 支持查看平台设定的会员折扣（只读），可分别设置平日、周末、节日的本店折扣（必须≤平台折扣） |
| 系统设置 | PMS对接 | 支持配置PMS系统对接参数，可测试连接并同步数据 |
| 系统设置 | 员工账号 | 支持添加、编辑、删除员工账号 |

## C端小程序

| 一级菜单 | 二级菜单 | 三级功能（用户故事） |
|---------|---------|-----------------|
| 用户中心 | 我的积分 | 支持查看积分余额，可查看积分流水记录和过期提示 |
| 用户中心 | 会员中心 | 支持查看当前会员等级，可查看会员权益说明和升级进度 |
| 用户中心 | 邀请好友 | 支持生成邀请链接或二维码，可分享给好友，查看邀请记录 |
| 酒店浏览 | 酒店列表 | 支持浏览酒店列表，可筛选和搜索酒店 |
| 酒店浏览 | 酒店详情 | 支持查看酒店详细信息、房型列表、设施展示、周边信息 |
| 酒店浏览 | 订单确认 | 支持填写入住人信息，可使用积分抵扣、选择积分换购服务，完成支付 |
| 订单管理 | 订单列表 | 支持按状态筛选订单（待付款/待入住/已入住/已离店/已完成/已取消），可查看订单 |
| 订单管理 | 订单详情 | 支持查看订单完整信息和费用明细，可取消订单 |
| 订单管理 | 支付成功 | 支持查看支付结果，可跳转查看订单详情 |
